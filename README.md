# Offensive Security Notes

A structured knowledge base documenting my pentesting learning journey, covering web application, network, Active Directory, and cloud attack techniques.

## Structure

| Folder | Content |
|---|---|
| `methodology/` | PTES and OWASP Testing Guide notes |
| `web-application/` | OWASP Top 10, Burp Suite, PortSwigger labs |
| `network-infrastructure/` | Nmap, Metasploit, service exploitation |
| `active-directory/` | Kerberos attacks, BloodHound, lateral movement |
| `cloud/` | AWS and Azure attack techniques |
| `ctf/` | HackTheBox, TryHackMe, VulnHub writeups |
| `tools/` | Custom scripts |
| `templates/` | Pentest report and writeup templates |

## Disclaimer

All content in this repository is for educational purposes only. Techniques documented here are practised exclusively in authorised lab environments.
